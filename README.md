# Tree diversity of Atlantic Forest remnants (Ramos et al.)

Reproducible R code and workflow for the manuscript **“Tree diversity of Atlantic Forest remnants: connectivity and size as drivers of floristic richness”** (Ramos et al.).
This repository contains scripts to process data, run statistical analyses, and reproduce figures and tables.

## Contents
- `data/` (raw and/or processed data; see notes on availability)
- `R/` or `scripts/` (analysis scripts)
- `figures/` (generated outputs)
- `tables/` (generated outputs)
